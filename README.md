# CTF数据可视化平台
  #
## 实现功能  
* 根据ip反向跟踪访问源信息，并实现数据可视化
* 使用图表展示各队对提交flag数据的可视化
* log监测与展示，一血提示，全站推送
* 反作弊警示，全站推送  
* 恶意访问请求ip，一键iptables  
* ... 

## 需求分析  
* 目前的大部分CTF平台后台对于访问数据分析的能力都比较弱，同时不只是主办方，选手对于比赛数据的观感也仅限于排行榜，因此我想对这方面进行改进，同时也能够对异常ip进行检测，保障平台安全。
